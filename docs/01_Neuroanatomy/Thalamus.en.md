# Thalamus

> *Thalamus is the cortex's "gateway"; nearly all sensory signals (except olfaction) pass through. Located in the diencephalon. ~ 50 nuclei: relay, association, intralaminar, reticular. LGN (vision), MGN (audition), VPL (somatosensory), VPM (face), MD (PFC interface). Damage → complex syndromes. Thalamocortical loop is a consciousness-hypothesis key.*
>
> **Difficulty**: Intermediate
> **Prerequisites**: [Cortex](Cortex.en.md), [Brainstem](Brainstem.en.md)

---

## 1. Location

- Deep central
- Either side of 3rd ventricle
- Above brainstem, below cortex
- Two-egg shape

---

## 2. Nucleus Classes

### 2.1 Specific Relay

- Sensory + motor relay
- 1-to-1 mapping
- Most cortical areas have corresponding thalamic nucleus

### 2.2 Association

- Interfaces with PFC, parietal high-order regions
- Many-to-many

### 2.3 Intralaminar

- Arousal, attention
- Connects with reticular system

### 2.4 Reticular

- Shell layer
- GABAergic, inhibitory
- Modulates thalamocortical oscillations

## 4. Thalamocortical Loop

```
Cortex (layer 6) ──→ Thalamus
       ↑                ↓
       └────────────────┘
       (layer 4)
```

- Reciprocal connection
- Oscillations: alpha, sleep spindles, delta
- Spindles key to sleep memory consolidation

---

## 5. Sleep Spindle

- NREM sleep 11-15 Hz brief burst
- Generated by reticular thalamic nucleus
- Linked to memory consolidation
- Reduced in schizophrenia

## 6. PyTorch — Thalamocortical Oscillator

```python
import torch

def thalamocortical_oscillator(T=1000, dt=0.5):
    """Wilson-Cowan-like loop."""
    E = 0.0  # cortex excitatory
    I = 0.0  # thalamic inhibitory
    history = []
    for _ in range(T):
        dE = (-E + 1.0 / (1 + torch.exp(torch.tensor(-(E - I + 0.5)))) ) * dt / 10
        dI = (-I + 1.0 / (1 + torch.exp(torch.tensor(-(E - 0.3)))) ) * dt / 10
        E += dE
        I += dI
        history.append((E.item(), I.item()))
    return history
```

## 7. Damage

- **Thalamic stroke**: ipsilateral hemiparesis, hemisensory loss, cognitive
- **Thalamic pain syndrome (Dejerine-Roussy)**: post-stroke chronic central pain
- **Fatal familial insomnia (FFI)**: PRNP mutation → bilateral anterior thalamus degeneration → total insomnia + death
- **Korsakoff syndrome**: thiamine deficiency → mid-anterior thalamus damage → confabulation + amnesia
- **Coma**: bilateral thalamic damage → loss of consciousness

---

## 8. Thalamus + Consciousness

- Bilateral lesion → coma indicates thalamus necessary for consciousness
- Crick & Koch: thalamocortical loop is NCC
- Llinás 2002: 40 Hz thalamocortical is consciousness substrate

---

## 9. AI Connection

- **Attention with relay** (like Transformer): some analogize to thalamus
- **Memory consolidation during sleep** similar to RNN spindle replay
- Modern AI has no explicit thalamus analog, but inspiration exists

## 10. Common Pitfalls

### 10.1 Thalamus = Sensory Relay

No; also PFC interface (MD), cerebellar relay (VL), attention, consciousness.

### 10.2 Olfaction Through Thalamus

Wrong; olfaction enters piriform directly.

### 10.3 LGN = Passive Relay

Actually: gain control, attention modulation.

### 10.4 Thalamus 100 Regions Are Modules

Some functionally overlap; not entirely modular.

### 10.5 Coma Solely Thalamus

Usually thalamus + brainstem RAS combined.
