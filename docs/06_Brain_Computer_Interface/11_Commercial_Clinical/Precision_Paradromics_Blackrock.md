# Precision、Paradromics、Blackrock

**Neuralink 和 Synchron 占据头条**，但三家**第二梯队公司**在 2025–2026 取得关键突破：**Precision Neuroscience**（Layer 7 FDA 510k 2025-03）、**Paradromics**（Connexus FDA 2025-11）、**Blackrock Neurotech**（BrainGate 长期合作）。这三家各自填补生态位，共同定义 BCI 商业格局。

## 一、Precision Neuroscience

### 公司背景

- 2021 Founded by Ben Rapoport（**前 Neuralink 联合创始人**）
- 纽约总部
- 融资 ~$135 M（到 2024）

### 产品：Layer 7 Cortical Interface

- **薄膜 ECoG**：厚度 **1/5 头发丝**
- **1024 电极**
- 通过**头骨小切口**插入（~1 cm）
- **不穿透脑组织**（比 Neuralink 更安全）

### 定位

"**比 Neuralink 安全，比 Stentrode 高分辨率**"

### 技术优势

- 柔性薄膜
- 大面积覆盖（多区域）
- 易取出 + 替换
- 可**双向**（读 + 刺激）

## 二、Precision 里程碑

### 2023 首次人体

- 暂时性植入（术中使用）
- 术后移除
- 验证安全 + 信号质量

### 2024 长期植入

- 首位长期植入患者
- 数月记录
- 信号质量良好

### 2025-03：FDA 510k 批准（K242618）

- **首家 1024 通道 BCI FDA 510k 通过**
- "临时植入"路径（30 天）
- **临床诊断用途**

### 2025 Q3–Q4 扩展

- 扩展到**长期植入**
- 10+ 患者队列
- 商业化路径建立

## 三、Paradromics

### 公司背景

- 2015 founded by Matt Angle
- 奥斯汀总部
- 融资 ~$100 M

### 产品：Connexus Direct Data Interface

- **1,000–10,000 电极**
- 每"**模块**" 420 微线
- **多模块**可并发植入
- 无线 + 有线版本

### 技术优势

- **最高通道密度**
- 分布式模块灵活
- 成熟电子设计

## 四、Paradromics 里程碑

### 2023 动物验证

- 猪、羊长期记录
- 稳定性 ≥ 6 个月

### 2024 人体准备

- IDE 申请
- 制造量产

### 2025-11：FDA Breakthrough + IDE 批准

- **官方首例人体植入**（预期 2025 底）
- 癫痫监测用途（首批）
- 扩展到 BCI 意图解码

### 2026 预期

- 小队列（3–5 患者）
- 完整功能验证
- 与 Neuralink 正面竞争通道数

## 五、Blackrock Neurotech

### 公司背景

- 2008 founded from Utah Array 技术
- **学术 BCI 研究标准供应商**
- 2023 被 **Tether / iCapital** 收购

### 产品：Utah Array 生态

- **NeuroPort Array**：研究用 Utah Array
- **MoveAgain**：临床版本（开发中）
- **Blackrock Cerebus**：数据采集系统

### 定位

- **BrainGate 长期合作方**
- 20 年植入经验
- **覆盖绝大多数学术 BCI 论文**

## 六、Blackrock 的 MoveAgain

### 设计

- 基于 Utah Array
- 单阵列 96 通道
- 直接 FDA 510k 路径

### 2021 FDA Breakthrough

- 优先审查路径
- 预期 2026-2027 批准

### 与学术共同推进

- BrainGate 2002+ 数据
- 已积累**数十位患者**经验
- **安全性数据最充分**

## 十、技术趋势

### 1. 通道数竞赛

- Neuralink、Paradromics、Precision 都向 10,000+ 通道前进
- 但**解码瓶颈** 不在通道数

### 2. 安全性提升

- Precision 表面、Synchron 血管
- **"不穿脑"** 成新范式

### 3. 全栈 vs 标准

- Neuralink 全栈
- Blackrock 标准（BrainGate 使用）
- 两种路径并存

### 4. AI 集成

- LLM + BCI（Synchron × OpenAI）
- 所有公司追赶

## 十二、逻辑链

1. **Neuralink + Synchron + Precision + Paradromics + Blackrock** 构成 2026 BCI 核心格局。
2. **Precision Layer 7** FDA 510k 2025-03——首家 1024 通道批准。
3. **Paradromics Connexus** FDA IDE 2025-11——高通道模块化。
4. **Blackrock** 学术金标准，MoveAgain 2026-27 批准预期。
5. **生态位分化**：各家定位不同适应症/路径。
6. **资本 + 并购** 预期 2027 医疗巨头进入。
7. **中国 Neuracle** 可能成为首个**商业化**侵入式 BCI——全球监管格局改变。
