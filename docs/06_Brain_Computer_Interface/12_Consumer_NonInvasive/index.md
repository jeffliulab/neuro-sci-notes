# 消费级与非侵入 BCI

侵入式 BCI 是医疗器械，面向少数患者；非侵入式 BCI 才有可能走向亿级用户。本章讨论三件事：当前消费级 EEG 设备的能力边界、Apple AirPods 等主流品牌的神经传感专利、以及"思维打字"这一长期愿景在 2024-2026 的现实状态。

**和第 11 章的关系。** 第 11 章是侵入式 BCI 的商业图景（医疗器械路径）；本章是非侵入式 BCI 的消费图景（消费电子路径）。两者监管完全不同：FDA / NMPA 对前者要求多年临床；对后者基本只要 FCC 电磁兼容与一般消费电子标准。两条路径的共同悖论——**信号质量与产品形态成反比**——是本章贯穿性的张力：要把电极藏进耳机 / 眼镜 / 头带，就必须接受比 Utah 阵列差几个数量级的信噪比，所以解码侧（第 04、05 章）必须做很多代偿。

**学习路径。** 「EEG 消费设备全景」先建立 **Muse / Emotiv / OpenBCI / Neurable** 的横向比较（研究 vs 消费、湿电极 vs 干电极、采样率与通道数）；然后用「Apple 神经传感专利」聚焦最有商业体量的玩家——AirPods EEG 已在 2023 年获专利授权，预期成为消费级 BCI 走向主流的关键产品；最后用「思维打字现状」做现实检验：为什么经过 20 年研究，**EEG 思维打字仍未实用**，侵入与非侵入端的 WER 差距如何，以及多模态（眼动 + EMG + EEG）是否能突破带宽瓶颈。
