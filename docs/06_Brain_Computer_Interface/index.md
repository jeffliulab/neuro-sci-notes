# 脑机接口

脑机接口（Brain-Computer Interface, BCI）是一个正在同时发生在神经科学、人工智能、精密制造、临床医学和宪法立法中的前沿领域。本章不是一本 BCI 入门教程，而是一份以 **"意图到行动（Intention-to-Action）"** 为主轴的系统化学习笔记。

**本章与[类人智能](https://jeffliulab.github.io/ai-notes/01_AI/03_Frontiers/index.md)并列。** 类人智能讨论的是"如何在数字侧构造心智"（预测编码、世界模型、因果、元学习），本章讨论的是"如何在生物神经与机器之间建立直接通路"。两者共同构成通向 AGI 与具身智能的两条并行路径。
