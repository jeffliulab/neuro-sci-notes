# 发展简史

BCI 的百年历史可以分为五个阶段。每一阶段的分界不是单个论文，而是一个让整个领域的 Overton Window 发生移位的标志性事件。

## 一、神经电信号的发现（1924–1970）

| 年份 | 事件 |
| --- | --- |
| 1924 | **Hans Berger** 首次记录人类 EEG，提出 α/β 节律的概念 |
| 1929 | Berger 发表《Über das Elektrenkephalogramm des Menschen》，EEG 从实验室走向临床 |
| 1950s | 神经生理学家定位**运动皮层**躯体图谱（Penfield homunculus） |
| 1969 | **Eberhard Fetz** 首次证明猕猴可以学会**自主调节**单个神经元放电率——BCI 概念的雏形 |

Fetz 1969 年的工作尤其具奠基性：他让猴子通过视觉反馈学会控制自己单个神经元的放电频率。这证明了"主动操控神经活动"在生物学上是可行的。

## 二、BCI 概念成型（1970–2000）

| 年份 | 事件 |
| --- | --- |
| 1973 | **Jacques Vidal** 在 *Annual Review of Biophysics* 上发表 "Toward direct brain-computer communication"，首次明确提出 BCI 概念 |
| 1988 | **Farwell & Donchin** 利用 P300 电位构建首个字母拼写 BCI |
| 1991 | **Jonathan Wolpaw** 在 Wadsworth Center 开始使用 sensorimotor μ 节律构建 EEG BCI |
| 1990s | **Andrew Schwartz** 在猴子上实现 **population vector** 基础的皮层内解码机械臂 |
| 1999 | **Chapin et al.** 在大鼠上实现闭环 BMI 控制机械臂喝水 |

这一阶段的关键是**学术共识**：BCI 不再是孤立的奇思妙想，而是一个拥有自己期刊、会议、研究范式的领域。

## 三、植入式 BCI 进入人体（2000–2015）

| 年份 | 事件 | 意义 |
| --- | --- | --- |
| 2004 | **Cyberkinetics / BrainGate** 获 FDA IDE，首次人体植入 Utah 阵列 | 商业公司首次进入临床 |
| 2006 | **Hochberg et al.** Nature 发表 BrainGate 首个患者（Matthew Nagle）用意念控制光标与机械手 | BCI 里程碑论文 |
| 2008 | **Velliste et al.** Nature 发表猴子用皮层解码自给自食 | 4 自由度机械臂自主取食 |
| 2012 | **Hochberg et al.** Nature：Cathy Hutchinson 用意念驱动机械臂喝咖啡 | 首个具身日常活动演示 |
| 2012 | **Gilja et al.** eLife 提出 **ReFIT-KF**，闭环自适应卡尔曼滤波 | 性能提升 2× |
| 2013 | **Collinger et al.** Lancet：7 自由度 "Pitt arm" 机械臂 | 高维度临床转化 |
| 2014 | **Nicolelis Walk Again Project** 在世界杯开幕式用 EEG-BCI 外骨骼踢球 | 全球科普事件 |
| 2015 | **Aflalo et al.** Science：Caltech Andersen 在后顶叶植入解码高层意图 | 意图解码新脑区 |

这一阶段的决定性转变是"**BCI 真的能在瘫痪患者身上用**"。Hutchinson 喝咖啡的视频在 YouTube 上的影响力让 BCI 从学术圈进入大众视野。

## 四、深度学习进入 BCI（2015–2023）

| 年份 | 事件 |
| --- | --- |
| 2016 | **Flesher et al.** Sci Transl Med：首次人类 ICMS 体感反馈 |
| 2018 | **Pandarinath et al.** Nat Methods 发布 **LFADS**——深度学习建模神经动力学 |
| 2018 | **Lawhern et al.** 发布 **EEGNet**——紧凑型 CNN 成为 EEG 解码基线 |
| 2019 | **Neuralink** 首次公开展示 N1 原型、1024 通道柔性线、手术机器人 |
| 2019 | **Anumanchipalli et al.** Nature：UCSF 语音合成 BCI 早期工作 |
| 2020 | **Ganzer et al.** Cell：**双向 BCI** 同一片皮层同时读运动、写触觉 |
| 2020 | **Synchron** 获 FDA IDE，经血管 Stentrode 进入临床 |
| 2021 | **Willett et al.** Nature：**90 字符/分钟手写 BCI** |
| 2021 | **Fernández et al.** Science Advances：皮层视觉假体 phosphene 感知 |
| 2021 | **Flesher et al.** Science：ICMS 反馈 2 倍加速机械臂任务 |
| 2022 | **Hong Bo（洪波）清华 NEO** 半侵入式 BCI 在患者上成功试用 |
| 2023 | **Willett et al.** Nature：**62 WPM 侵入式语音 BCI** |
| 2023 | **Metzger et al.** Nature：UCSF 数字 avatar 同步表情 + 语音 |
| 2023 | **Tang et al.** Nat Neurosci：fMRI 语义解码 |
| 2023 | **MindEye** ICLR：fMRI → 图像重建 |
| 2023 | **Meta Défossez et al.** Nature MI：非侵入 MEG 语音解码 41% 识别率 |

这一阶段的关键词是 **端到端深度学习 + 闭环 + 双向**。解码器从线性高斯模型被 CNN / LSTM / Transformer 取代；BCI 从"遥控机械臂"升级为"具身感觉反馈"；非侵入 BCI 在 LLM 加持下首次逼近实用。

## 五、商业化临界点（2024–2026）

| 年份 | 事件 |
| --- | --- |
| 2024-01 | **Neuralink** 首例人体植入（Noland Arbaugh） |
| 2024-04 | **Shanechi lab** 发布 **DPAD** 算法，行为相关 vs 无关神经活动分离 |
| 2024-09 | **Synchron COMMAND** 完成 6 人临床试验，主终点达标 |
| 2024 | **MindEye2** ICML：1 小时 fMRI 数据即可重建图像 |
| 2024-08 | **UC Davis** 语音 BCI 97% 准确率 |
| 2024 | Neuralink 第 2、3 例植入（Alex、Brad） |
| 2025-03 | **Precision Neuroscience** Layer 7 获 FDA 510(k) 批准 |
| 2025-05 | 基础模型 **POYO+、Neuroformer、NDT3** 相继发布 |
| 2025-11 | **Paradromics** 获 FDA 批准进入临床试验 |
| 2026-03 | **Neuracle × 清华** NEO 半侵入式 BCI 获中国 NMPA 批准——**全球首款商业化侵入式 BCI** |
| 2026 | Neuralink 植入数达到 12+（多国扩展） |

2024–2026 的核心变化是**"多个正交突破在同一个时间窗口发生"**：
- **信号端**：Precision Layer 7 把"无须开颅"带入临床
- **算法端**：神经基础模型把跨被试迁移变成现实
- **商业端**：中国 NMPA 首批、美国多家并行
- **立法端**：科罗拉多、明尼苏达、UNESCO 神经权利框架陆续建立

## 六、平行的立法时间线

BCI 技术时间线和立法时间线从 2021 年开始同步推进：

| 年份 | 立法事件 |
| --- | --- |
| 2021 | 智利通过**宪法修正案**，将"大脑活动及其产生的信息"列为受保护的个人数据 |
| 2024 | **科罗拉多** 修订隐私法纳入"neurodata" |
| 2024-05 | **明尼苏达** 通过"**认知自由法**"（首个州级专门立法） |
| 2024 | **UNESCO** 通过《神经技术伦理建议》起草工作 |
| 2025 | **EU AI Act** 正式生效，对神经接口提出专门要求 |

这份平行时间线说明：**BCI 不是单纯的技术叙事，而是一个同时涉及工程、商业、宪法、国际治理的复合叙事**。这是本章必须兼顾 AI/工程/临床/伦理四个面的根本原因。

## 七、逻辑链

1. **BCI 不是一个新概念**——Berger 1924 的 EEG、Fetz 1969 的自主神经元调节已经奠定技术基础。
2. **BrainGate 2006 的 Matthew Nagle 是分水岭**——植入式 BCI 第一次证明能在瘫痪患者身上工作。
3. **2016 之后深度学习 + 闭环双向 BCI 成为主旋律**——LFADS、EEGNet、Willett 手写/语音、ICMS 反馈。
4. **2023 年的 Willett 语音 + MindEye + Meta 非侵入解码是 LLM × BCI 融合的起点**。
5. **2024-2026 是商业化临界点**——FDA/NMPA 多家获批；全球首款商业化侵入式 BCI 在 2026 年 3 月由中国落地。
6. **立法时间线与技术时间线同步推进**，神经权利不再是未来问题。
