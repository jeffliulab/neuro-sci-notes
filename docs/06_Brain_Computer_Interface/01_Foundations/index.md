# 基础概念

本章给出 BCI 领域的顶层分类、发展简史与核心术语。它是全章的入口，也是后续所有章节的词汇表。

**为什么从这里开始？** 不同 BCI 系统在性能、风险、目标人群上差异极大，把它们一起讨论会产生大量混淆。**侵入性**（侵入 / 微创 / 非侵入）、**信号方向**（解码 / 编码 / 双向）、**应用场景**（临床 / 消费 / 科研）是把任何一个 BCI 工作快速归位的三把钥匙——后续章节里出现的 Neuralink、Stentrode、Muse、MindEye 都可以放进这个三维表里读。

**学习路径。** 完全没有背景的读者建议按顺序阅读三节：先用「BCI 概述与分类」建立空间感，再用「发展简史」把 Berger（1924 EEG）→ Kennedy（1998 第一例侵入）→ BrainGate（2006）→ Neuralink（2024 PRIME）→ Neuracle（2026 NMPA 首批）这条 100 年时间线串起来；最后用「核心术语与指标」掌握 **ITR / bitrate / 闭环 / decoder vs encoder** 等贯穿全章的关键概念，再进入第 02 章神经生理学基础。
